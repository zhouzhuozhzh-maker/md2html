# FitRank Team MVP

一个本地静态 SaaS 原型，用于团队记录体测、饮食摄入、运动消耗，并按周期生成排行榜。

## 使用方式

直接在浏览器打开：

```text
fitness-saas/index.html
```

首次打开需要注册一个本地用户。数据暂存在浏览器 `localStorage`，适合产品原型、内部评审和交互验证。

## 已支持

- 每个用户独立上传体测信息：体重、体脂、腰围、备注。
- 注册本地用户并进入个人工作台。
- 拍照记录摄入食品，并填写餐次和热量。
- 上传 Apple 运动截图，或手动填写运动项目、时长和消耗大卡。
- 按天、周、月、季度生成小组排行榜。
- 通过用户 ID 搜索并加入健身小组。
- 激励小游戏：每日挑战抽签、完成挑战得分、热量护城河反馈。

## 后续 SaaS 化方向

- 接入账号系统和真实组织租户。
- 增加后端数据库，替换 `localStorage`。
- 引入图片 OCR 或大模型识别食品/Apple 运动截图。
- 增加管理员后台、团队邀请、赛季规则和导出报表。
